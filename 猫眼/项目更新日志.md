# 项目更新日志

## 2026-01-05

### ✨ 新增功能

### 🐛 问题修复
1. **修复爬虫失效问题**
   - **问题现象**：点击爬取按钮后，返回结果显示"成功爬取 0 部电影"。
   - **修复方案**：切换至移动端接口并优化 JSON 解析逻辑。
2. **修复数据自动加载问题**
   - **问题现象**：页面初次加载时会显示旧有的爬取数据，影响用户体验。
   - **修复方案**：移除了 `DOMReady` 时的自动请求逻辑，确保只有在用户点击"开始爬取"后才加载和显示数据。

### 💄 界面优化
1. **现代化 UI 设计**

2. **交互体验提升**
   - 添加了加载状态提示，爬取过程中按钮显示加载动画。
   - 增加了操作结果的 Toast 提示消息（成功/失败/警告）。
   - 优化了表格样式，支持鼠标悬停高亮。

     - 包含动态旋转和鼠标视角跟随效果，提升了页面的视觉深度和高级感。
   - **3D 沉浸式 UI 设计 (2026-01-05 更新)**
     - **全透明背景融合**：移除了所有主要容器（卡片、表格、JSON展示区）的背景色，实现了文字与 3D 粒子背景的完美融合。
     - **3D 透视交互**：为页面容器添加了 CSS 3D 透视 (`perspective`)，并实现了跟随鼠标移动的全局 3D 倾斜效果 (`rotateX/Y`)，增强沉浸感。
     - **文字融合**：增强了所有文字元素的阴影 (`text-shadow`)，确保在动态背景下依然清晰可读。

### 🌟 视觉效果增强 (2026-01-05 深夜更新)
1. **Three.js 背景光影升级**
   - **动态光照系统**：引入了环境光 (`AmbientLight`) 和平行光 (`DirectionalLight`)，并添加了环绕场景运动的动态点光源 (`PointLight`)，实现了丰富的光影变化。
   - **阴影渲染**：开启了 WebGL 渲染器的阴影贴图 (`ShadowMap`) 支持，配置了柔和阴影 (`PCFSoftShadowMap`)，使物体之间产生真实的遮挡关系。
   - **3D 几何体浮动**：将原有的简单粒子点阵升级为漂浮的 3D 几何实体（正方体、八面体、四面体），这些实体使用标准材质 (`MeshStandardMaterial`)，能够响应光照并产生高光和阴影，极大地增强了场景的体积感和真实感。

## 2026-01-06

### ✨ 新增功能：3D 镭射影廊
1. **沉浸式 3D 展厅**
   - 新增独立页面 `/gallery`，构建了一个基于 WebGL (Three.js) 的环形 3D 电影海报墙。
   - 实现了海报的环绕式布局和鼠标交互：悬停时海报放大并显示详细信息（评分、片名）。

2. **高级视觉特效：镭射卡面 (Holographic Shader)**
   - 编写了自定义 GLSL 着色器 (Shader)，为每张海报添加了**动态镭射反光效果**。
   - 实现了**菲涅尔效应 (Fresnel Effect)**，使卡片边缘随视角变化产生强烈的高光。
   - 实现了**彩虹色散**，模拟真实镭射卡的七彩光泽，随鼠标视角转动而流转。

### 🔧 核心优化
1. **图片跨域代理 (Image Proxy)**
   - 新增后端接口 `/api/image_proxy`，解决浏览器端直接请求猫眼图片遭遇的 CORS 跨域限制，确保 WebGL 纹理能正确加载。

2. **数据容灾机制**
   - 优化了数据获取逻辑：优先使用实时爬取数据 -> 尝试自动重新爬取 -> **自动降级至模拟数据**。
   - 内置了一套包含真实海报链接的高质量模拟数据，确保在网络不通或反爬限制即使触发时，用户仍能完整体验 3D 影廊的视觉效果。

3. **导航优化**
   - 在主控面板新增"3D 影廊展示"快捷入口，实现了从数据爬取到可视化展示的无缝跳转。

### 🌟 影廊极致进化 (2026-01-06 晚)
1. **真实海报映射**
   - 实现了 3D 卡面的**异步纹理映射**，现在每个电影卡片都会显示抓取到的原始海报封面。
   - 配合 **Canvas 混合渲染**：在海报下方自动叠加科技感渐变遮罩，确保文字信息在复杂图片背景下依然清晰，保持了整体的赛博朋克美感。

2. **核心健壮性修复**
   - 彻底修复了由于脚本异步加载顺序导致的 `ReferenceError` 报错。
   - **模板热更新**：启用了 Flask 的 Debug 模式，并配置了 `v2.2` 缓存穿透参数，确保用户能立即看到最新的 UI 修改，解决了严重的浏览器缓存滞后问题。

3. **中心视觉强化**
   - 重构了中央 3D 装置，将其升级为**超大型发光三棱锥体**，配合 StandardMaterial 的自发光属性，使其在影廊正中心成为了一个不可忽视的视觉焦点。
